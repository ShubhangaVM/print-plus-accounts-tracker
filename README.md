# Print Plus Accounts Tracker

A lightweight business accounting tracker built using **Python and Flask**.

This application was created to help manage a small printing shop's daily transactions such as sales, purchases, pending payments, and reports.

## Features

- Dashboard with business summary
- Sales tracking
- Purchase tracking
- Customer management
- Seller management
- Pending payment tracking
- Monthly and yearly reports
- Invoice generation
- Search functionality
- Desktop executable build

## Tech Stack

- Python
- Flask
- SQLite
- HTML / CSS
- PyInstaller (for desktop app)

## How to Run

1. Install dependencies, pip install -r requirements.txt
2. Run the application, python app.py
3. Open browser, http://127.0.0.1:5000

## About the project
This project was built as a practical tool for managing real buisness records digitally instead of manual bookkeeping.
